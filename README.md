# RainbowPanels
パネルを3つ並べて消す、カラフルパズルゲーム

[▶ ゲームを遊ぶ](https://isomissm.github.io/rainbowpanels-game/rainbowpanels.html)

## 遊び方
パネルをタップして入れ替えて、同じ色を3つ以上並べてパネルを消していくよ<br>
連鎖でゲージが貯まって、BURSTでお邪魔なパネルを消せる！<br>
スコアが上がると新しいパネルが出現するようになるよ<br>

## 特殊パネル
- 🌈 虹パネル：周りのパネルを巻き込んで一気に消去
- 🌑 闇パネル：放置すると危険！早めに消そう
- ⚡ 雷玉：お邪魔パネルと闇パネルを一掃
- ☀️ 晴玉：5×5マス内にあるお邪魔(雲)パネルが赤パネルに変わるよ
- 🌧️ 雨玉：ランダムでパネルが青パネルに変わるよ
- ⚪️ 雪玉：放置するとランダムでパネルが凍ってしまう！
- 🔴🟠🟡🟢🔵🟣 7色の玉：全部集めるとレベルレインボー発動
