# ナナイロパネル - Rainbow Panels
パネルを3つ並べて消すカラフルパズルゲーム

[▶ ゲームを遊ぶ](https://isomissm.github.io/rainbowpanels-game/rainbowpanels.html)

## 遊び方
クリック／タップでパネルを入れ替えて同じ色を3つ以上並べてパネルを消そう<br>
連鎖でゲージが貯まって、BURSTでお邪魔なパネルを消せる！<br>
スコアが上がると新しいパネルやモードが出現するようになるよ<br>

## 特殊パネル
- 🌈 虹パネル：周りのパネルを巻き込んで一気に消去
- 🌑 闇パネル：放置すると危険！早めに消そう
- ⚡ 雷玉：お邪魔パネルと闇パネルを一掃
- ☀️ 晴玉：5×5マス内にあるお邪魔(雲)パネルが赤パネルに変わるよ
- 🌧️ 雨玉：ランダムでパネルが青パネルに変わるよ
- ⚪️ 雪玉：放置するとランダムでパネルが凍ってしまう！
- 　　七色の玉：7つ集めると…!?
- ⭐️ スコアが増えると「夜モード」に変わるよ！流れ星を3つ集めよう

## その他の機能
- オプションでパネルアニメーション速度、アイコンの種類、七色の玉の形を変更できるよ
- パネル画像を自分の好きな画像に差し替えられるよ
- オートセーブ機能（ウィンドウを閉じても再開時に続きからプレイできるよ）
